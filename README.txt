How to use:
1. Download and unzip the folder.
2. Open index.html in a browser to view locally.
3. To publish online:

   - OR deploy to Vercel: sign up at vercel.com, import the repository, and deploy (it's a static site).
Editing:
- Change texts directly in the HTML files or update via the small data-* attributes for RU/KZ.
- Add your photos by replacing the placeholder divs with <img src="path"> and uploading images to the repo.
